Software Development

My main area of interest in programming is desktop and back end web development using the Microsoft .NET framework. C# is my language of choice when I need to develop a custom solution. It is a powerful language with a robust base class library and awesome features like LINQ, garbage collection, and asynchronous programming.

I am also an Angular developer, and have made this my front end framework of choice for web applications. The provided command line tools, intergrated testing framework, and being built around TypeScript make this a framework with which I greatly enjoy working.

Sometimes I just don't need the overhead of the massive .NET framework, and Python is another favorite language. It is perfect language for automating repetitive tasks or processing large batches of data. Python is also what I strongly recommend as a first lanaguge to learn for programming beginners.

I bring all these areas of interest together in my current role as a Software Engineer at Republic Services in Scottsdale, Arizona.
