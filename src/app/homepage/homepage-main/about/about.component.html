Software Development

My main area of interest in programming is desktop and back end web development using the Microsoft .NET framework. C# is my language of choice when I need to develop a custom solution. It is a powerful language with a robust base class library and awesome features like LINQ, garbage collection, and asynchronous programming.

Sometimes I just don't need the overhead of the massive .NET framework, and Python is another favorite language. It is the perfect language for me to automate repetitive tasks or process large batches of data. Python is also what I strongly recommend as a first lanaguge to learn for programming beginners.

I am also an Angular developer, and have made this my front end framework of choice for developing web applications. The provided command line tools, intergrated testing framework, and being built around TypeScript make this a framework with which I greatly enjoy working.

I bring all these areas of interest together in my current role as a Software Engineer at Republic Services in Scottsdale, Arizona. See my resume for more information about my professional experience.

Projects

C# Async Demo As a Software Engineer, I like to mentor team members and help others understand programming concepts. This mini project was created to provide a basic, practical demonstration of an asynchronous programming pattern in C# and highlight the performance gains that can be achieved. I also wrote a blog post that provides an in-depth walk through of this code and the task-based asynchronous pattern in .NET.

SignalR Demo This repository contains another mini project which is intended demonstrate the capabilities of SignalR, and provide guidance on how to implement it in a web application built with .NET Core 3.1 and Angular 10. I use SignalR in applications I have developed for work in order to provide real-time communication between the client and server, but I found the documentation to be a little bit lacking. I was also having some dificulty finding thorough examples of how to use SignalR, so I wrote my own tutorial with a simple implementation of SignalR to share with other developers. A working example of an application which uses SignalR to update a progess bar on the client here.

MB Utils This project is a MapBasic module and .NET library that extended MapInfo programming functionality by exposing .NET and Win32 API calls as easy to use MapBasic functions and subs. This was the first library I ever wrote and was a great learning experience in how to document, release, and support an open source software project, albeit on a small scale. I have long since ended work on this project, but I'm sharing it here to show how my career and code quality has progressed.
